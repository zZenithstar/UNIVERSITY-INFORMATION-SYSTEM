# UNIVERSITY-INFORMATION-SYSTEM
Java-based console School Management System with Admin, Teacher, and Student roles. Includes login, profile management, and CRUD operations using text-file storage.



A console-based School Management System developed in Java using Object-Oriented Programming principles. Built because apparently humanity enjoys managing chaos through menus.

---Features---
Multi-role login system (Admin / Teacher / Student)
Add, update, delete users
Profile viewing system
Text-file based data storage
Menu-driven console interface
Separate management for Students, Teachers, and Admins
Organized class structure with encapsulation

---Technologies Used---
Java
OOP Concepts (Encapsulation, Classes, Methods)
File I/O (BufferedReader, FileWriter, FileReader)
IntelliJ IDEA

---Project Purpose---

This project was created to improve Java programming skills, file handling knowledge, and Object-Oriented Programming logic through a real-world management system simulation.
